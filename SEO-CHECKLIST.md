# 搜索引擎收录部署清单

## 📋 部署后操作清单

### 一、验证网站可访问

- [ ] 访问 `https://taoheal.pages.dev` 确认网站正常运行
- [ ] 访问中文版本 `https://taoheal.pages.dev/zh`
- [ ] 访问英文版本 `https://taoheal.pages.dev/en`
- [ ] 验证sitemap可访问: `https://taoheal.pages.dev/sitemap.xml`
- [ ] 验证robots.txt可访问: `https://taoheal.pages.dev/robots.txt`

### 二、Google Search Console 收录

- [ ] 访问: https://search.google.com/search-console
- [ ] 添加资源（域名或URL前缀）
- [ ] 选择"网址前缀"方式：`https://taoheal.pages.dev`
- [ ] 验证网站所有权（HTML标记、Google Analytics、DNS记录等）
- [ ] 提交sitemap: 在"网站地图"中提交 `/sitemap.xml`
- [ ] 提交主页URL到"网址检查"工具请求编入索引
- [ ] 检查"覆盖范围"报告，确认页面被收录

### 三、Bing Webmaster Tools 收录

- [ ] 访问: https://www.bing.com/webmasters
- [ ] 登录Microsoft账户
- [ ] 添加网站: `https://taoheal.pages.dev`
- [ ] 验证网站所有权
- [ ] 提交sitemap
- [ ] 提交URL进行爬取

### 四、百度搜索资源平台收录（可选）

- [ ] 访问: https://ziyuan.baidu.com
- [ ] 登录百度账户
- [ ] 添加网站并验证所有权
- [ ] 提交sitemap
- [ ] 提交URL收录

### 五、其他搜索引擎（可选）

- [ ] DuckDuckGo: https://duckduckgo.com/submit_url
- [ ] Yandex Webmaster: https://webmaster.yandex.com
- [ ] SoGou搜索资源平台
